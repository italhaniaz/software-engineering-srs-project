# Software Engineering SRS Project

## Overview
This repository contains the Software Requirements Specification (SRS)
for a Paper Generator Web Application developed as part of the
Software Engineering semester project.

## Contents
- Introduction
- Overall Description
- Functional Requirements
- Non-Functional Requirements
- UML Diagrams (Use Case, Class, Sequence, Activity)

All UML diagrams are created using the tikz-uml package in LaTeX.

## Tools Used
- LaTeX
- tikz-uml package
